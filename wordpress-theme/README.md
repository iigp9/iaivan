# Tema WordPress iaivan

Tema moderno de IA con diseño oscuro futurista y efectos de neón cyan para WordPress.

## Instalación

1. Descarga el tema completo
2. Sube la carpeta `wordpress-theme` a `/wp-content/themes/` en tu instalación de WordPress
3. Renombra la carpeta a `iaivan` si lo deseas
4. Activa el tema desde Apariencia > Temas en tu panel de WordPress

## Personalización

Este tema es totalmente personalizable desde el panel de WordPress:

### Desde Apariencia > Personalizar encontrarás:

1. **Sección Hero**
   - Título Hero
   - Subtítulo Hero

2. **Solución Integral**
   - Título
   - Descripción
   - Texto del botón
   - URL del botón

3. **Header CTA**
   - Texto del botón del header
   - URL del botón

4. **Plan Gratis**
   - Título
   - Precio
   - Descripción
   - Texto del botón
   - URL del botón

5. **Plan Profesional**
   - Título
   - Precio
   - Descripción
   - Texto del botón
   - URL del botón

6. **Plan Enterprise**
   - Título
   - Precio
   - Descripción
   - Texto del botón
   - URL del botón

### Menús

Desde Apariencia > Menús puedes crear y asignar menús a:
- Menú Principal (aparece en el header)
- Menú Footer (aparece en el pie de página)

### Logo

Desde Apariencia > Personalizar > Identidad del sitio puedes:
- Cambiar el logo
- Modificar el nombre del sitio
- Cambiar el lema
